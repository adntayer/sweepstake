Heatmap: Pontos por dia
Cada coluna é um dia, cada linha é um jogador (ordenado do melhor para o pior). A cor varia de ■ vermelho (0%) a ■ verde (100%) conforme o percentual de pontos que o jogador fez em relação ao máximo teórico daquele dia (nº de jogos × 15 pts). A última linha mostra esse máximo teórico.
Role horizontalmente se a tabela não couber na tela.
Como verificar se os números estão certos?
  • Max possível = número de jogos do dia × 15 pts (placar exato). Ex: 3 jogos = 45 pts máx.
  • Média pts/dia = soma dos pontos de todos os jogadores ÷ número de participantes.
  • Aproveitamento = média dos percentuais individuais. Para cada jogador: pts dele ÷ max teórico, depois tira-se a média entre todos.
  • Para validar: pegue um dia, para cada jogador calcule % = pts dele ÷ max teórico, depois tire a média. O resultado deve bater com o "Aproveitamento".
